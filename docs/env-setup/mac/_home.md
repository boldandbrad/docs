# macOS Setup

Setup a new macOS environment.

## Install Tools

### Xcode CLI tools

```zsh
xcode-select --install
```

### Homebrew

Install [Homebrew](https://brew.sh) :beer: package manager.

```zsh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```
