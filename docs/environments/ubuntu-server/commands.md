# Commands

## Networking

Get internal IP Addresses (v4 & v6):

```bash
hostname -I
```

Get external IP Address (v4):

```bash
curl v4.ident.me
```

Get external IP Address (v6):

```bash
curl v6.ident.me
```

Get current network devices and interfaces:

```bash
ip link
```

## Power

Restart server:

```bash
sudo reboot now
```

Shutdown server:

```bash
sudo shutdown now
```
