# Server Management

Commands for managing the minecraft server.

## Service Commands

- Status:

    ```bash
    systemctl status minecraft
    ```

- Start:

    ```bash
    systemctl start minecraft
    ```

- Stop:

    ```bash
    systemctl stop minecraft
    ```

- Restart

    ```bash
    systemctl restart minecraft
    ```
